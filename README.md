## Requironments
Download and install [MelonLoader](https://melonwiki.xyz/#/README)

## Instalation
Download [latest release](https://github.com/Flyover-ArtSk/SparkInTheDarkMapMod/releases/latest) and unpack all content to `%GAMEFOLDER%/Mods`

## Usage
In game press M (for keyboard) or Right Stick (for gamepad) to open map
